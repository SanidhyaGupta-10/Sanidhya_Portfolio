# Portfolio

![Portfolio](/images/portfolio.png)

## Description

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react) uses Babel (or oxc when used in rolldown-vite) for Fast Refresh.
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses SWC for Fast Refresh.

## Tech Stack

- **React** – UI library
- **Vite** – Build tool & dev server
- **JavaScript** (or **TypeScript** – optional) – Language
- **ESLint** – Linting
- **Babel** or **SWC** – Transpilation / Fast Refresh
- **Git** – Version control

## Features

- Hot Module Replacement (HMR) for instant UI updates
- Fast Refresh via Vite plugins
- Ready-to-use ESLint configuration
- Easy switch to TypeScript with `vite-template-react-ts`

## Installation

```bash
git clone <repository-url>
cd Actual-Portfolio
npm install
npm run dev
```

Open your browser at `http://localhost:5173` to see the app.

## Usage

- Edit files in `src/` to start building your portfolio.
- Add more images to `public/images/` and reference them in your components.
